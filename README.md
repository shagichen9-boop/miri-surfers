# Miri Surfers: NIS Dash 3D

A Subway Surfers-style endless runner built with Three.js. Dodge buses (Egged, Dan, Kavim, Metropoline, Superbus), barriers, and low-bridge signs while collecting NIS coins on the streets of Israel.

## How to Play

- **Arrow keys / WASD** - Move left/right, jump (up), slide (down)
- **Swipe** on mobile - Same controls via touch gestures
- Collect NIS coins to increase your score
- Pick up powerups: Magnet (attracts coins) and x2 (double NIS)
- Watch the bus ticket price climb as you collect more coins

## Files

| File | Description |
|------|-------------|
| `index.html` | Full standalone game (single file, daytime theme) |

## Running Locally

Serve the folder with any static HTTP server:

```bash
python3 -m http.server 8000
```

Then open http://localhost:8000/
